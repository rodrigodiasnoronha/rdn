Antes de começar, verifique se você atendeu aos seguintes requisitos:

## 🚀 Instalando

Para instalar o projeto, siga estas etapas:

- Instale as dependências do projeto:

```bash
    $ npm i
```
- Execute o projeto:

```bash
    $ npm run dev
```

- O projeto deve ser executado na em `localhost:3000`
