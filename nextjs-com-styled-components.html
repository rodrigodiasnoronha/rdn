Next.js + Styled Components - Como configurar seu projeto

MAI 13, 2020

Tutorial de como utilizar Next.js com Styled Components

Rodrigo Dias NoronhaMAI 13, 2020

Neste artigo irei demonstrar como criar um projeto e utilizar o Styled Components junto ao Next.js, um Framework web baseado em React. Lembrando que o único requisito para iniciar este tutorial será ter o Node.js instalado em sua máquina.

Iniciando o projeto

Para criar um projeto com o Next.js, você pode utilizar a própria CLI do Next.js ou criar o projeto de maneira manual (na qual será abordada neste artigo). Também é possível iniciar um projeto pelo starter do próprio Next.js.

Então, vamos começar. Primeiro será necessário criar uma pasta para nosso projeto:

  mkdir nextjs-com-styled-components

Em seguida, entrar na pasta:

  cd nextjs-com-styled-components

Inciaremos o NPM em nosso projeto agora:

  npm init -y

Agora, iremos instalar as dependências do nosso projeto. Para isto, será necessário apenas 4 dependências:

  npm i react react-dom next styled-components

Como o próprio Next.js possui seu próprio bundle incluso em sua biblioteca, então não precisaremos necessariamente configurar quase nada para dar um start em nosso projeto. O próprio Next.js já trás tudo configurado e pronto pra você meter a mão no código.

Continuando, para verificar se tudo está funcionando, adicionaremos as 3 linhas seguintes dentro de scripts no nosso package.json, pois elas são necessárias para darmos inicio no servidor de desenvolvimento do Next.js.

  {
    "name": "nextjs-com-styled-components",
    "version": "1.0.0",
    "description": "",
    "main": "index.js",
    "scripts": {
        "test": "echo \"Error: no test specified\" && exit 1",
        "start": "next start", // Adicione essa linha
        "dev": "next dev", // Adicione essa linha
        "build": "next build" // Adicione essa linha
    },
    "keywords": [],
    "author": "",
    "license": "ISC",
    "dependencies": {
        "next": "^9.4.0", 
        "react": "^16.13.1", 
        "react-dom": "^16.13.1",
        "styled-components": "^5.1.0"
    }
}

Agora, para verificar se esta tudo funcionando, crie uma pasta chamada pages com um index.js dentro do diretório raiz do nosso projeto. Você pode prencher com um Componente do React ou como eu exemplifico abaixo:

import React from 'react';

const Home = () => <h2>Hello, World</h2>;

export default Home;

Agora, para iniciar nosso servidor de desenvolvimento, digite no terminal:

npm run dev

O servidor do Next.js será aberto automaticamente em localhost:3000 (tenha certeza de ter executado o npm run dev dentro do diretório raiz do projeto). Espere ele iniciar e, então, ele deve mostrar o seguinte em seu navegador:

Mas apenas sair injetando o estilos com o Styled Components ao nosso projeto não será uma boa ideia (ainda), já que ele não será "pré-processado" corretamente pelo Next.js. Para isso precisamos configurar o nosso projeto para dar suporte ao Styled Components.

Para isso, precisaremos baixar a seguinte dependência:

npm i babel-plugin-styled-components -D

Adiante, precisamos configurar o bundle do Next.js para utilizar a dependência que instalamos em nosso projeto, para isso, precisaremos criar um arquivo .babelrc na raiz do nosso projeto e adicionar o seguinte conteúdo ao arquivo:

{
    "presets": ["next/babel"],
    "plugins": [
        [
            "styled-components",
            {
                "ssr": true
            }
        ]
    ]
}

Além disso, precisamos fazer mais um passo. Precisamos criar um arquivo chamado _document.js dentro do nosso diretório pages. Este arquivo lida com o próprio template HTML e sua configuração dentro do Next.js. Iremos adicionar o seguinte conteúdo a este arquivo:

import Document from 'next/document';
import { ServerStyleSheet } from 'styled-components';

export default class MyDocument extends Document {
    static async getInitialProps(ctx) {
        const sheet = new ServerStyleSheet();
        const originalRenderPage = ctx.renderPage;

        try {
            ctx.renderPage = () =>
                originalRenderPage({
                    enhanceApp: (App) => (props) =>
                        sheet.collectStyles(<App {...props} />),
                });

            const initialProps = await Document.getInitialProps(ctx);
            return {
                ...initialProps,
                styles: (
                    <>
                        {initialProps.styles}
                        {sheet.getStyleElement()}
                    </>
                ),
            };
        } finally {
            sheet.seal();
        }
    }
}

Agora basta reiniciarmos nosso servidor do Next.js e tudo estará configurado para usarmos o Styled Components. Se quiser testar nossa página index, acrescente o seguinte código à ela:

import React from 'react';
import styled from 'styled-components';

const Container = styled.div`
    h2 {
        color: #e02041;
        text-align: center;
        margin: 15px;
    }
`;

const Home = () => (
    <Container>
        <h2>Hello, World</h2>
    </Container>
);

export default Home;
